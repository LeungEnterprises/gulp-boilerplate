# Gulp Boilerplate
Boilerplate for making a static site using Gulp.

## Instructions
`git clone https://github.com/LeungEnterprises/gulp-boilerplate`

`cd gulp-boilerplate`

`npm install && bower install`

`gulp dev`

## Deploy
Edit your details in the `deploy.sh` script.
